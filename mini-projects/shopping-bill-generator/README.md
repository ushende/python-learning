# Shopping Bill Generator

A simple command-line Python program that generates a shopping bill based on user input.

## Features

- Enter customer name
- Select quantities for multiple products
- Calculate the total bill
- Apply discounts automatically
  - 5% discount for bills ₹500 or more
  - 10% discount for bills ₹1000 or more
- Display a formatted final bill

## Concepts Used

- Variables
- User Input
- Arithmetic Operators
- `if` / `elif` / `else`
- Basic Output Formatting

## Run

```bash
python main.py
```
