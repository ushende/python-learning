# Shopping Bill Generator

A simple Python program that generates a shopping bill based on the products and quantities entered by the user.

## Features

* Takes customer name as input
* Calculates the total bill
* Applies a discount based on the total amount
* Displays the final bill in a clean format

## Concepts Used

* Variables
* User Input
* Arithmetic Operators
* `if`, `elif`, `else`
* Basic Output Formatting
